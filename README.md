# Chatbot-for-phone-BD

# Это программа чатбот для простейшей базы данных
# Что умеет программа ?

# Программа запускается командой /start
 

![Стартовый экран](https://github.com/EvgenyBusleiko/Chatbot-for-phone-BD/blob/main/Start.jpg)

# Показывать содержимое  БД - команда /all


![Стартовый экран](https://github.com/EvgenyBusleiko/Chatbot-for-phone-BD/blob/main/all.jpg)

# Добавлять новые записи - команда add/


![Стартовый экран](https://github.com/EvgenyBusleiko/Chatbot-for-phone-BD/blob/main/Add.jpg)

# Удалять записи - команда /delete


![Стартовый экран](https://github.com/EvgenyBusleiko/Chatbot-for-phone-BD/blob/main/delete.jpg)

# Искать запись как по Фамилии, так и по номеру телефона - команда /find

![Стартовый экран](https://github.com/EvgenyBusleiko/Chatbot-for-phone-BD/blob/main/find.jpg)

# Закончить работу с программой можно командой /stop

![Стартовый экран](https://github.com/EvgenyBusleiko/Chatbot-for-phone-BD/blob/main/stop.jpg)
